Semantic HTML

What is Semantic HTML?

Semantic HTML refers to the use of HTML markup that conveys meaning about the content it contains.

Semantic HTML helps search engines and browsers understand the structure and meaning of web content.

It improves accessibility for users with disabilities and enhances the overall user experience.

Some examples include header, nav, article, footer

Common Semantic Elements

  • <header>

    • Header elements are used to define the header section of a document or a section.

  • <nav>

    • Navigation elements are used to define a set of navigation links.

  • <article>

    • Article elements are used to define independent, self-contained content.

  • <footer>

    • Footer elements are used to define the footer section of a document or a section.

  • <section>

    • Section elements are used to define a thematic grouping of content.

  • <aside>

    • Aside elements are used to define content that is tangentially related to the main content.

  • <figure>

    • Figure elements are used to encapsulate media content, such as images or diagrams, along with their captions.

  • <figcaption>

    • Figcaption elements are used to provide a caption for the content within a figure element.

Benefits of Semantic HTML

  • Improved SEO
  • Better Accessibility
  • Enhanced User Experience
